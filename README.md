# Personal agent skills

Standardized skills for setting up repositories and enforcing shared engineering habits.

## Skills

| Skill | Purpose |
| --- | --- |
| [`setup`](skills/setup/) | Initialize a repo with baselines (docs, lint/test, hooks, CI, optional release/publish) |
| [`engineering-principles`](skills/engineering-principles/) | Shared coding principles and CONTRIBUTING guidance |
| [`wiki`](skills/wiki/) | Init / update / ask against an in-repo wiki, or install CI that runs update |

## Usage

Install into a project via [skills.sh](https://skills.sh):

```bash
npx skills add https://github.com/manojbajaj95/skills
```
